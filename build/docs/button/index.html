Button

A customizable button component with automatic text contrast and hover effects.

Basic Usage

<script>
    import { Snap } from 'snap-ui';
</script>

<Snap.Button>Click me</Snap.Button>

Props

PropTypeDefaultDescription
bgColourstring"ffffff"Background color as hex string (without #)
idstring"button"HTML id attribute for the button
marginnumber16Margin around the button in pixels
onClick(event: MouseEvent) => void() => {}Click event handler
childrenSnippetRequiredButton content (text, icons, etc.)

Examples

Basic Button

<Snap.Button>Default Button</Snap.Button>

Custom Colors

<Snap.Button bgColour="2563eb">Blue Button</Snap.Button>
<Snap.Button bgColour="dc2626">Red Button</Snap.Button>
<Snap.Button bgColour="16a34a">Green Button</Snap.Button>

Dark Colors with Text Auto-Contrast

<Snap.Button bgColour="1f2937">Dark Gray</Snap.Button>
<Snap.Button bgColour="7c2d12">Dark Brown</Snap.Button>
<Snap.Button bgColour="581c87">Dark Purple</Snap.Button>

With Click Handler

<script>
    function handleClick(event) {
        alert('Button clicked!');
        console.log('Click event:', event);
    }
</script>

<Snap.Button 
    bgColour="f59e0b" 
    onClick={handleClick}
>
    Click for Alert
</Snap.Button>

Custom Spacing

<Snap.Button margin={8}>Small Margin</Snap.Button>
<Snap.Button margin={24}>Default Margin</Snap.Button>
<Snap.Button margin={40}>Large Margin</Snap.Button>

Behavior

Automatic Text Contrast

The component automatically calculates the optimal text color (black or white) based on the background colorâ€™s relative luminance, ensuring proper readability and accessibility.

Hover Effects

All buttons include a subtle brightness reduction on hover (hover:brightness-90) for better user interaction feedback.

Border Styling

The component automatically generates a darker border color based on the background color for enhanced visual depth.
