Card

A flexible container component with customizable dimensions and automatic border styling.

Basic Usage

<script>
    import { Snap } from 'snap-ui';
</script>

<Snap.Card>
    <h3>Card Title</h3>
    <p>Your content goes here.</p>
</Snap.Card>

Props

PropTypeDefaultDescription
widthstring"100"Width of the card (adds “px” if not specified)
heightstring"100"Minimum height of the card (adds “px” if not specified)
bgColourstring"ffffff"Background color as hex string (without #)
prosestring"false"Enable prose styling for rich text content (“true” or “false”)
childrenSnippetRequiredCard content

With Prose Styling

Article Title

When prose is enabled, the card applies rich text formatting that makes your content look great with proper typography spacing and styling.

  • Lists are styled nicely
  • Text has proper spacing

Behavior

Automatic Border Color

The component automatically generates a darker border color based on the background color by reducing RGB values by 30, creating subtle visual depth.

Responsive Width

Cards are responsive with max-w-[90vw] on mobile and max-w-full on larger screens, ensuring they don’t overflow on small devices.

Flexible Height

The height prop sets a minimum height, allowing the card to grow with content while maintaining a baseline size.
