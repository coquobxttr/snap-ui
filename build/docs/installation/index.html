Installation

How to install dependencies and structure your app

Package Manager

npm install snap-ui

Tailwindcss

SnapUI uses Tailwind v4.1 with the typography and forms plugins. See the website for a more detailed tutorial on how to install.

npm install -D @tailwindcss/typography
npm install -D @tailwindcss/forms

Configure tailwindcss

Link tailwind to the package components

@import 'tailwindcss';
@plugin '@tailwindcss/forms';
@plugin '@tailwindcss/typography';
+ @source "../node_modules/snap-ui/dist/components";
